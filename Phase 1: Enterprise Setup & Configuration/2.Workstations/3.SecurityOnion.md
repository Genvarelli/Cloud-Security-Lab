### What is Security Onion?

**Security Onion** is a free, open-source platform for:
- **Network Security Monitoring (NSM)**
- **Log Management**
- **Intrusion Detection & Response**

Think of it as the **Kali Linux** of network monitoring‚Äîpreloaded with powerful tools:

- **Zeek (formerly Bro)** ‚Äì Network traffic analysis.
- **Suricata** ‚Äì Intrusion detection and prevention.
- **Elastic Stack** ‚Äì Log management and analysis via Elasticsearch, Logstash, and Kibana.

#### üì° Network Security Monitoring (NSM)

#### üõ°Ô∏è Incident Response

Security Onion strengthens an organization's cybersecurity posture by providing:
